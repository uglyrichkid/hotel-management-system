# Hotel Management System

A full-stack hotel management system designed for managing hotels, rooms, guests, bookings, payments, users, and reports.

## Description

This project is an enterprise-style Hotel Management System that helps hotel staff manage daily operations through a web-based admin panel.

The system supports hotel management, room management, guest records, bookings, payments, reports, and user access control.

## Features

- Hotel management
- Room management
- Guest management
- Booking management
- Payment tracking
- Dashboard and reports
- User and role management
- JWT-based authentication
- Multi-hotel support
